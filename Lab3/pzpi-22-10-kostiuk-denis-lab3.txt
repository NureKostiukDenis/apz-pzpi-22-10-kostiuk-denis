1. Мета роботи 
     Розробити front-end частину програмної системи у вигляді web-застосунку, яка включає інтерфейс для адміністрування та взаємодії користувачів із системою. Реалізувати фільтрації, сортування та управління даними.
     
2. Інженерні рішення 
2.1. Використання React Admin:
     Для побудови адміністративної панелі використано бібліотеку react-admin, яка дозволяє швидко створювати CRUD-інтерфейси з використанням готових компонентів (Datagrid, List, Filters, Edit, Create тощо).
2.2. REST API:
     Комунікація з бекендом відбувається через REST API з використанням simpleRestProvider. Для цього було модифіковано серврену частину для роботи з simpleRestProvider за стандартом наданим в react-admin.
2.3. Фільтрація та сортування:
     Реалізовано фільтри по імені та секціях товарів через FilterList, FilterLiveSearch, useListContext, а також сортування та пагінацію.
2.4. Управління складом:
     Адміністратор має можливість створювати, редагувати, переглядати та фільтрувати товари, призначати секції тощо.
2.5. Обробка помилок:
     Реалізовано виведення повідомлень про успішні операції або помилки за допомогою useNotify.
